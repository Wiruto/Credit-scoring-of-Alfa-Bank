Кредитный скоринг Альфа банка

Для корректной работы сервера необходимо придерживаться следующийх правил:

  • Сервер должен быть запущен из корневой директори файла server.py;
  • Исходные данные для предсказания должны быть представлены в табличной форме, описанной в разделе Data
    сорeвенования Kaggle от Альфа Банка;
  • Таблица с исходными данными должна быть сохранена в формате csv.


Для выполения кредитного скоринга клиентов банка необходимо выбрать файл с исходными данными.
Пример таблицы для выполнения предсказания можно скачать в разделе Data соревнования на платформе Kaggle
или на GitHub репозитория автора блендинга моделей Wiruto.
Сохраните файл с исходными данными в любой директории компьютера и выбретите его с помощью формы ниже.
